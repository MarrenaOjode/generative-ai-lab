# Generative AI Lab

This repository contains solutions for the Generative AI for Data Science lab assignment completed in Jupyter Notebook.

## Project Overview

The lab focuses on using Generative AI tools to assist with:
- Data analysis
- SQL query generation
- Debugging analytics code
- Prompt engineering
- Data visualization

## Scenarios Included

### 1. Retail Inventory Analysis
- Inventory turnover calculations
- Slow-moving item detection
- Stockout prediction
- Business visualizations using Python

### 2. Website Analytics Debugging
- Debugging engagement metrics
- Fixing bounce rate calculations
- Session time validation
- Data cleaning and validation
- Analytics visualizations

### 3. Customer Segmentation Query
- SQL customer segmentation
- High-value customer identification
- Active user filtering
- User preference analysis
- Optimized SQL using CTEs and window functions
